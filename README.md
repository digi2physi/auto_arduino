# DaVinci Context

> Ready-to-use Arduino component templates for beginners

## What is this?

A collection of tested, documented templates for common Arduino components. Each template includes:

- **Clear contract**: What inputs are needed, what outputs you get
- **Working code**: Tested with actual hardware
- **Easy customization**: Change only what you need

## Quick Start

1. Find your component in `components/`
2. Copy the `template.ino` to your project
3. Adjust the `USER CONFIGURATION` section
4. Upload and run
